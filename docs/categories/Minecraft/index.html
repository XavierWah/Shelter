recorderBot 使用说明

此为版本草图, 插件仍处于预开发周期.
你可以在此处找到项目的 Github 仓库.

简介

recorderBot 是由 XavierWah 开发用于 MCDReforged 的插件. 此插件基于 carpet 的 /player 命令进行拓展, 加入了记录信息, 查询历史记录等功能, 辅助玩家在服务器中更好的使用 Bot 协作.
此使用说明适用于 recorderBot v1.0-alpha .
